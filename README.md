# Homebrew Tap

This is the official Homebrew tap for [tool3](https://github.com/tool3) projects.

# Installation

```bash
brew tap tool3/tap
```

# Available Formulas

| Formula   | Description                                            | Install                            |
| --------- | ------------------------------------------------------ | ---------------------------------- |
| shellfie  | Transform terminal output into beautiful SVG images    | `brew install tool3/tap/shellfie`  |
| dvd       | Transform terminal output into beautiful animated SVGs | `brew install tool3/tap/dvd`       |
| chartscii | Transform terminal output into beautiful charts        | `brew install tool3/tap/chartscii` |
